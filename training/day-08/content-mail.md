# ターゲット企業向けコンテンツドラフトを作成する：アプローチメール案（claude）

---

## ▍人事部長向け

### 件名（3パターン）

| # | 件名 | 狙い・手法 |
|---|------|------|
| A | 離職した社員の、本当の退職理由 | 「本当の」という言葉で隠された情報への好奇心を刺激。自社に当てはまるかもと思わせる |
| B | 評価制度より先に、直すべきものがあります | 逆説型。「制度は整えたのに…」と感じている担当者の注意を引く |
| C | 人事部長が一番、損をしている | 意外性のある主語で「なぜ？」を誘発。自分ごとに感じさせる |

### 本文

```
○○株式会社 人事部長 ××様

突然のご連絡失礼いたします。
Sirius（シリウス）の▲▲と申します。

「評価基準はあるのに、マネージャーによって結果がばらつく」
このお悩みを抱える人事部長様に、
AIアセスメントで評価を定量化・言語化するSiriusをご紹介しております。

導入企業では評価への納得度が1年で38%→71%に改善、
離職率の低下にもつながっています。

サービスの詳細につきましてお話させていただきたく、15分だけお時間をいただけますでしょうか。
ご都合のよい日程を折り返しご案内いたします。

Sirius 営業担当 ▲▲
```
**文字数：約200字**

## ▍代表取締役 / COO向け

### 件名（3パターン）

| # | 件名 | 狙い・手法 |
|---|------|------|
| A | 組織が150人を超えると、起きやすいこと | 自社の規模感と重なる数字で「うちのことかも」と思わせる |
| B | 優秀な人が「辞める会社」と「残る会社」の違い | 対比型。経営課題として普遍性があり、どのフェーズの経営者にも刺さりやすい |
| C | 社長が知らない、現場の本音 | 「知らない情報がある」という不安を刺激。情報の非対称性を直接体感させる |

### 本文

```
○○株式会社 代表取締役 ××様

Sirius（シリウス）の▲▲と申します。

成長期のSaaS企業では、現場の細かいパフォーマンスが
経営層に届かなくなる「情報の非対称性」が起きがちです。

Siriusはアセスメント×AIで人材データを可視化し、
抜擢・配置・異動の意思決定をデータドリブンで支援します。

サービスの詳細につきまして、15分のオンラインでご説明できれば幸いです。
ご都合をお聞かせいただけますでしょうか。

Sirius 営業担当 ▲▲
```
**文字数：約190字**

## 📝 使い分けの考え方

3パターンとも**サービス名・会社名・ソリューションワードを件名に入れない**のがポイントです。開封前に「営業メール」と判断されるとそれだけで閉じられます。まず開封させ、本文で初めてSiriusを登場させる構成にしています。

- **好奇心ギャップ型**（A）：答えを伏せることで「続きが読みたい」を引き出す
- **逆説・対比型**（B）：「えっ、どういうこと？」という認知的不協和を使う
- **不安刺激型**（C）：「自分が知らない何かがある」という感覚を利用する
