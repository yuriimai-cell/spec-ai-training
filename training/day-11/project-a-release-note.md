# Project A リリースノート

## プロジェクト名
Saas製品競合比較ツール

## 公開URL
[https://project-n2nj2aczv-yuriimai-cells-projects.vercel.app/app]

## 概要
[何ができるツールか、1-2文で]

BtoB SaaSの商談準備で使える、AI競合比較表ジェネレーターです。 自社と最大3社の競合を入力すると、共通フォーマットの比較表を生成します。

## 主要機能
1.製品情報入力フォーム
2.AI比較表生成
3.送信プロンプト（ソース）表示

## 使用技術
- Next.js
- Tailwind CSS
- Vercel（ホスティング）
- AI（Cursor + Claude/ChatGPT）

## 開発プロセスの振り返り
- かかった時間: 約6時間
- AIにどれくらい頼ったか: 8~9割
- 自分で修正した箇所: UIのデザイン、要件定義（ツール使用後に表示される競合との比較表において、有用性を⚪︎や×などの記号で表す仕様をテキストで表す仕様へと変更した）
- 苦労した点: Vercelでのアップロードがうまくいかなかった。原因としてはアプリのファイルが src/app/app/ に入っていて /app ルートになっていた（別の階層に.tsxファイルが入っていた）ことだったが、その特定と対処までのAIとのラリーが多くなってしまった。
